>The Banking Simulation project is a comprehensive software application designed to mimic the core functionalities of a banking system.
>This project aims to provide a realistic and interactive experience allowing users to manage bank accounts, perform transections,and track their financial activities.
>Users can create,manage,and delete bank accounts,including checking and savings accounts.
>The system allows users to deposite, withdraw and transfer money between accounts.
>Users can check their account balances.
>This project is made in The Python language and used MySQL database management system to store account information records.
>A user-friendly interface is designed using GUI frameworks Tkinter.
>We need internet connection to work this project successfully.
>You need to replace app password of your gmail account in EmailHandler.py file.
